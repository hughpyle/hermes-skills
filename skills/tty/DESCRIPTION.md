This turned into Teletype Agent Weekend.

The ASR33 was the original terminal for Unix. The first ASCII machine. Mechanical keyboard and printer, a legacy of the telegraph age.

I dusted it off and printed some Katherine Nash and Frederick Hammersley artworks to show at @the_clark "Analog Day" (https://events.clarkart.edu/detail/5223).

Local loopback is cool, but I wanted to put tthe machine online.  So I put @NousResearch Hermes Agent on a Raspberry Pi and built a shell interface to chat with.


If you want to experience the hardware at home (sans oily smells), I ported my emulator to Hermes too.

This is quite a special terminal emulator, because it features very realistic printing sounds, recorded with an Ambisonic microphone.  The hum, keypresses, space and character printing, carriage return and the bell.  Two modes: lid open, lid closed.

Try it!
